# Hostel-Management-System
A system built for hostel room allocation for our College(IIT GUWAHATI) as a part of DBMS Course.

ER Diagram contains six attributes
1. Hostel
2. Hostel Manager
3. Rooms
4. Students
5. Applications
6. Messages
